## 1. 实现一个```new```操作符

### 实现逻辑

1. 它创建了一个全新的对象。

2. 该新对象会被接入传入函数的原型链中。

3. 它使fn中的```this```指向新创建的对象。

4. 如果函数没有返回对象类型 Object(包含 Functoin,Array,Date,RegExg,Error)，那么 new表达式中的函数调用将返回该对象引用。

### 代码
```javascript
function New (fn) {
  var obj = {};
  if (fn.prototype !== null) {
    obj.__proto__ = fn.prototype;
  }

  var ret = fn.apply(obj, Array.prototype.slice.call(arguments, 1));
  if ((typeof ret === 'object' || typeof ret === 'function') && ret !== null) {
    return ret 
  }
  return obj
}
```

## 5. call

### 实现逻辑

1. 检查是否被函数调用

2. 检查传入this是否正确

3. 将待调用函数赋值给context（最好配合symbol）

4. 调用然后删除该函数

### 代码

```javascript
Function.prototype.call = function(context, ...args) {
  if (typeof this !== 'function') {
    throw new TypeError('Error')
  }

  context = context || window;

  const fn = Symbol();
  context[fn] = this;

  context[fn](...args);
  delete context[fn];
}
```

## 7.bind

### 实现逻辑

bind其实就是call中的函数被返回了而已

这里因为bind是返回一个函数

所以要考虑返回的函数是否能被new操作符使用的情况

### 代码
```javascript
Function.prototype.bind = function(context) {
  if(typeof this !== 'function') {
    throw new TypeError('Error')
  }

  const that = this
  const args = [...arguments].slice(1)

  return function F(){
    if(this instanceof F) {
      return new that(...args, ...arguments)
    }

    return that.apply(context, args.concat(...arguments))
  }
}
```
